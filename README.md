# 📚 Repositório Central de Estudos

Bem-vindo ao meu repositório central de aprendizado! 

Este espaço foi criado com o objetivo de armazenar, versionar e organizar todos os meus códigos de estudos, resoluções de exercícios, anotações e experimentos práticos. Tudo aquilo que é focado no meu desenvolvimento contínuo e que não constitui um projeto de software fechado fica guardado aqui.

## 🗂️ Como este repositório está organizado?

Para manter o ambiente limpo e isolar os diferentes contextos de aprendizado, optei por organizar este repositório utilizando **Branches**. Cada ramificação (`branch`) representa uma frente de estudo específica.

Atualmente, o repositório está dividido nas seguintes branches:

*   **`main`**: A branch principal (onde você está agora), que serve como guia e contém este README explicativo.
*   **`ufba`**: Atividades, trabalhos e práticas acadêmicas da Faculdade de Computação da UFBA.
*   **`ford-enter`**: Códigos, exercícios e anotações relacionados ao programa Ford Enter.
*   **`codeforces`**: Minhas resoluções para os desafios de lógica e programação competitiva do Codeforces.
*   **`leetcode`**: Soluções e práticas de algoritmos e estruturas de dados do LeetCode.

> **Nota:** Este é um repositório vivo. Novas branches serão adicionadas futuramente conforme eu explore novas tecnologias, cursos ou plataformas de estudo.

## 🧭 Como navegar pelos códigos

Como os arquivos estão separados por branches em vez de pastas, você não verá os códigos de estudo diretamente na página inicial. 

**Pelo GitHub:**
Vá no botão dropdown escrito `main` (no canto superior esquerdo, logo acima da lista de arquivos) e selecione a branch da área de estudo que você deseja visualizar.

**Pelo Terminal (Localmente):**
Se você clonar este repositório, pode visualizar e navegar entre as áreas de estudo usando os comandos:

```bash
# Baixar todas as referências de branches do repositório remoto
git fetch all

# Listar todas as frentes de estudo disponíveis
git branch -a

# Mudar para a branch de estudos que deseja acessar (ex: ufba)
git checkout ufba
